# La Gran Competencia de Ribs - Casa de Kira 🍖🔥

Sitio web oficial para la competencia de costillas de cerdo entre los hermanos:
* **Keyla Daniela** (La Maestra del Fuego)
* **José Alejandro** (El Guardián del Humo)
* **Juan José** (El Señor de la Brasa)

## Características
* **Countdown:** Temporizador hasta el 25 de mayo de 2026.
* **Perfiles Interactivos:** Conoce la historia y técnica de cada guerrero.
* **Ubicación Dinámica:** Mapa interactivo de la Casa de Kira.
* **Confirmación de Asistencia (RSVP):** Formulario para los invitados.
* **Efectos Visuales:** Costillas voladoras y brasas animadas para una experiencia inmersiva.

## Tecnologías
* HTML5 Semántico
* CSS3 (Vanilla) con animaciones personalizadas
* JavaScript (Vanilla) para lógica y efectos
* Google Fonts (Cinzel & Inter)
* Google Maps Embed API
